# Workflow: Share Reports Package

## Objective

Generate a shareable package of all weekly audit reports that can be easily distributed to colleagues via email or file sharing.

## When to Use

Use this workflow when you need to:
- Share the latest weekly reports with stakeholders
- Provide a complete archive of all analysis reports
- Create a self-contained package with navigation

## Prerequisites

- Weekly reports must already exist in `output/` (generated by `generate_week_report.php`)
- Run this after completing the weekly analysis cycle

## Steps

### 1. Generate the Shareable Package

From the `projects/audit-stats/` directory, run:

```bash
php tools/generate_shareable_package.php
```

### 2. Verify Output

The tool produces two files:

| File | Description |
|------|-------------|
| `output/index.html` | A navigation page listing all reports with links to HTML, PDF, and Markdown versions |
| `output/audit-stats.zip` | A zip archive containing all output files (21 files, ~600KB) |

### 3. Share with Colleagues

**Option A: Share the ZIP file**
- Send `output/audit-stats.zip` via email or file sharing
- Recipients can extract and open `index.html` in any browser
- All reports are self-contained and work offline

**Option B: Share the index.html directly**
- If sharing via a web server, upload the entire `output/` directory
- Share the URL to `index.html`
- Colleagues can browse reports and download PDFs as needed

## What Gets Included

The package includes:
- All weekly report files (`.html`, `.pdf`, `.md`)
- The `summary.json` cross-week summary
- The `index.html` navigation page
- Token usage log (`.token_log.jsonl`)

The following are excluded:
- `.gitkeep` (placeholder file)
- The `audit-stats.zip` file itself (can't zip itself)

## Tips

- **Re-run anytime**: Safe to re-run — it overwrites previous outputs
- **File size**: The zip is typically 500KB-1MB, suitable for email
- **Offline viewing**: All HTML and PDF files work without internet access
- **Browser compatibility**: index.html uses modern CSS but works in all current browsers
